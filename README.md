Walkthrough video: https://www.loom.com/share/f43661e61ad44562b94ebebd5953d91a

## 📌 Tech Stack

- React (18+)
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router

## ⚙️ Setup Instructions

### Clone the repository

```bash
git clone https://github.com/Honeshwar/fenrir-assessment.git
cd fenrir-assessment
```

### Install dependencies

```bash
npm install
npm run dev

watch at: http://localhost:5173
```

### Build for production

```bash
npm run build
npm run preview
```
